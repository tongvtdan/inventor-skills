## Invocation

```
/inventor-skills:innovate
/inventor-skills:innovate Our B2B SaaS project management tool for remote teams
/inventor-skills:innovate A coffee shop loyalty program
/inventor-skills:innovate Electric scooter sharing service
```

Apply the SIT (Systematic Inventive Thinking) methodology to generate breakthrough innovation concepts for: $ARGUMENTS

Follow the SIT skill instructions exactly. Work through all five thinking tools — Subtraction, Multiplication, Division, Task Unification, and Attribute Dependency — applying each to the components and attributes of the subject provided.

Produce a complete **Innovation Brief** with:
1. A system map (components + attributes)
2. Concepts generated per tool (at least 2 per tool)
3. A ranked top-3 to top-5 innovation concepts with feasibility assessment
4. Recommended next steps for validation

If $ARGUMENTS is empty, ask the user what product, service, or business they want to innovate before starting.
